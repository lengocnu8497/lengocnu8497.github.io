How to customize font in React Native?

Before doinganything, I want to clarify that there are other ways of doing this task that I probably haven't known of. If you're reading this and do know the "less work" approach, please shoot me a message below, thanks ;)

STEP 1

Go to Google Fonts Site to download your desired font.
For my case, the font I want to use is Montserrat. Did you know, as I am typing this, 6.26 billion is the number of times Google Fonts API has served Montserrat on websites all over the world :allhailMontserrat:.

STEP 2

After extracting the file into your desinated folder, it is recommended that an assets folder is created to contain your extracted file so that the default path to get to your font folder is yourProject/assets/fonts/
Also, get the post-script format of the names or rename them to your liking.

STEP 3

Inside node modules folder, find react-native.config.js.
Under module.export, add this line:
assets: ['./assets/fonts/']

Some people suggested using rnmp link into package.json; however, the next release of React Native will no longer support that configuration.

STEP 4

In your terminal run react-native link. This will finish linking and copy the font in your fonts folder.

STEP 5

You will need to run npx react-native run-android in terminal for the font to be added and the project to be rebundled.

STEP 6
